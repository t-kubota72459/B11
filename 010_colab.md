## 2. セルの基本操作
Colabには2種類の「セル」があります。
* **コードセル**: Python のプログラムや計算式を書いて実行する場所。  
* **コメント**: `#` から後ろはコメントで計算には影響しません。  
* **テキストセル**: 説明やレポートの文章を Markdown で書く場所。

**実行方法**:
コードを入力した後、**`Shift` + `Enter`** を押すと計算が実行され、下に結果が出ます。
以下を実行してみましょう：

  ```python
  # 2 の 10 乗を求める
  2 ** 10
  ```

  ```python
  # 日本語も使えるよ
  # 文字データは " (ダブルクォーテーション) で括ります
  "日本語も使えるよ"
  ```

> 💡 電卓では 2 の 10 乗は [2] > [$y^x$] > [10] [=] と押します。
 
複数の式を連続して入力したときは、結果が表示されるのは最後の式だけです：

  ```python
  1+2
  3+4
  5+6 # 11 しか表示されません。
  ```

途中の結果も見たいときは `print()` を使うと確認できます。

### 解説：Markdown（マークダウン）とは？

モノづくりの設計やプログラミング、AI の学習を始めるにあたって、避けて通れないのが「ドキュメント（メモやレポート）作成」です。そのための強力なツールが **Markdown** です。

#### 1. 一言でいうと？
**「簡単な記号を使って、見やすい文書を素早く作るための書き方」** です。

Wordのようにマウスでカチカチと装飾（太字にしたり見出しにしたり）するのではなく、キーボードから手を離さずに、テキストだけで文書の構造を組み立てることができます。
#### 2. なぜ必要なの？

* **プログラミングとの相性が抜群：** Python などのコードを、実行結果と一緒にきれいに整理して保存できます。
* **AI・IT業界の標準：** エンジニアが使う GitHub や、技術系メモサイトのほとんどが Markdownを採用しています。
* **レポート作成の効率化：** 100分（2時限分）の長い実習内容を、構造的に（見出し、箇条書き、コード引用など）素早くまとめるのに最適です。

#### 3. 書き方の例

特別なソフトは不要です。メモ帳でも書けますが、特定の記号を入れることで意味を持ちます。

* **見出し：** `#` の数でレベルを変えます（`# タイトル`, `## 章` など）  
  `#` の数が多いほど、より細かい (小見出し) になる。
 
* **箇取り書き：** `-` や `*` を先頭に置きます。
* **コード：** バッククォート（\`）3つで囲むと、プログラムコードとして認識されます。

  ```python
  print("Hello, Hi-TC!")
  ```

## 3. 数式の入力ルール（Python記法）
数学の書き方と少し違う「エンジニアの書き方」を覚えましょう。

| 演算 | 数学の書き方 | Pythonでの書き方 | 実行例 |
| :--- | :--- | :--- | :--- |
| **加算** | $1+2$ | `1 + 2` | `3` |
| **減算** | $5-3$ | `5 - 3` | `2` |
| **乗算** | $4\times2$ | `4 * 2` | `8` |
| **除算** | $10\div3$ | `10 / 3` | `3.333...` |
| **べき乗** | $2^3$ | `2 ** 3` | `8` |
| **カッコ** | $2\times(3+4)$ | `(2 * (3 + 4))` | `14` |

> **注意！**：数学では $2a$ のように掛け算の記号を省略しますが、Pythonでは必ず `*` を書く必要があります（例：`2 * a`）。

## 4. 接頭語（単位）と指数の入力
工業数学で扱う「ミリ」や「マイクロ」は、`e` を使って入力します。

* **$10^3$ (k: キロ)** → `1e3` （1,000のこと）
* **$10^{-3}$ (m: ミリ)** → `1e-3` （0.001のこと）
* **$10^{-6}$ ($\mu$: マイクロ)** → `1e-6` （0.000001のこと）

> **注意！**：単位の **m** (ミリ) と **M** (メガ) は別物です。  
> Pythonでは小文字の `e` を使って `1e3` のように書くのが一般的です。

**練習コード（セルにコピーして実行してみよう）**:

  ```python
  # 0.1μF（マイクロファラド）を表示してみる
  c = 0.1e-6
  print(c)
  ```
