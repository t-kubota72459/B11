# 🛠️ エンジニア計算サバイバル・チートシート（完全版）

## 1. 【最重要】「右肩」から脱出せよ！
関数電卓の **[ $y^x$ ]** ボタンを押すと、カーソルが**右肩（指数）**に上がります。
そのまま入力を続けると、計算がすべて右肩に乗ってしまい、答えが狂います。

| 計算したい式 | 電卓の打ち方（EL-509T） | 操作のコツ |
| :--- | :--- | :--- |
| **$5^{3 \times 5}$** | `5` **[ $y^x$ ]** `3` `[✕]` `5` | ずっと右肩で計算するならそのままでOK。 |
| **$5^{3} \times 5$** | `5` **[ $y^x$ ]** `3` **`[ → ]`** `[✕]` `5` | **右矢印で「右肩」から抜けて**から `[✕]` ！ |

> **Pythonでの書き方：** $5^3$ は `5 ** 3` 。 $5^{3 \times 5}$ なら `5 ** (3 * 5)` と書く。

## 2. 困った時の「脱出」コマンド
「画面が変！」「エラーが出た！」という時は、迷わずこれ。

* **電卓がパニック！**: **`[HOME]`** ボタン（一撃で標準モードに戻る）
* **入力を全部消したい**: **`[ON/C]`** ボタン
* **Pythonが止まらない**: 実行ボタンの **「停止（■マーク）」** を押す

## 3. 電卓 vs Python：道具の使い分け

| やりたいこと | 関数電卓 (EL-509T) | Python (Colab) |
| :--- | :--- | :--- |
| **分数** ($\frac{2}{3}$) | **`[ a/b ]`** ボタン | `2 / 3` |
| **累乗** ($a^b$) | **`[ y^x ]`** ボタン | `a ** b` |
| **ルート** ($\sqrt{x}$) | **`[ √ ]`** ボタン | `math.sqrt(x)` |
| **巨大な数** ($10^{10}$) | **`[exp]`** ボタン | `5e10` |
| **微小な数** ($10^{-3}$) | **`[exp]`** → **`[(-)]`** `3` | `1e-3` |
| **直前の答え** | **`[ ANS ]`** ボタン | `_` (アンダースコア) |

## 4. エンジニアの「計算三原則」
1. **π（パイ）は「ボタン」で打つ**
   - `3.14` は禁止。電卓なら **`[ π ]`** 、Pythonなら **`math.pi`** 。精度を落とさない！
2. **「途中のメモ」は禁止**
   - 長い数字を紙に書いて丸め直さない。 **`[ ANS ]`** や **変数** に覚えさせる。
3. **丸めるのは「最後」だけ**
   - 答えを出す瞬間までフル精度で計算する。（指定がなければ小数点第1位まで）

## 5. 単位の読み替え（接頭語）
電卓の **`[ ENG ]`** ボタンで、単位を瞬時に切り替えられます。

* **$10^3 = \text{k}$ (キロ)** ： 1,000
* **$10^6 = \text{M}$ (メガ)** ： 1,000,000
* **$10^{-3} = \text{m}$ (ミリ)** ： 0.001
* **$10^{-6} = \mu$ (マイクロ)** ： 0.000001
