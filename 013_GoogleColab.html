Google Colab とは？

正式名称は「Google Colaboratory（コラボラトリー）」。

  • Googleが提供する無料のサービス。
  • ブラウザ（Chromeなど）を開くだけで、本格的なプログラミング（Python）ができる。
  • 自分のパソコンの性能は関係なし！計算はGoogleのスーパーコンピュータがやってくれる。

  • Googleで「Google Colab」と検索。

使い方②：枠（セル）にコードを書く

Colabの画面には、横長の「枠」があります。
これをセル（Cell）と呼びます。

この枠の中に、Pythonの命令文を書きます。

↓ プログラムを書くための「枠」＝ セル
print("Hello, World!")

使い方③：再生ボタンで実行！

  • セルの左側にある再生ボタンを押す。（または Shift + Enter）
  • その枠のプログラムだけが動く。
  • 結果はすぐ下に出現！

失敗しても、何度でも書き直して再生ボタンを押すだけです。

a = 10
b = 20
print(a + b)
30

使い方④：テキスト（メモ）も書ける

Colabには「コード」を書く枠と、「テキスト」を書く枠の2種類があります。

  • テキストセル: 実験の目的、考察、メモをきれいに書ける。
  • コードセル: 実際の計算や処理。

プログラムと実験ノートを「1つのファイル」にまとめられるのが最強！

実験1：データの可視化

収集したデータをグラフにして確認します。以下のコードを実行してください。

import matplotlib.pyplot as plt
plt.plot([1, 2, 3], [4, 5, 6])

数学が苦手？ だからこそColab！

全自動 計算＆グラフ機

面倒な計算や、ややこしい数式を自分で解く必要はありません。

「このデータでグラフを描いて」「平均値と分散を出して」とPythonにお願いするだけ。
数学は苦手でも、"計算ツール" を使いこなせればOKです！ 

# データを渡すだけでグラフになる！
import matplotlib.pyplot as plt

# 実験データ
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

plt.plot(x, y)
plt.show() # グラフ出現！

まとめ：Colabは現代の文房具

  • 面倒な準備ゼロ！ ブラウザを開くだけでスタート。

  • 計算やグラフ化は全部パソコンに丸投げ！
