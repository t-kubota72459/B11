# 第2回 指数表記と変数：エンジニアの数値作法

## 今日のゴール
* **3桁区切りの指数表記（e表記）**　を使いこなし、大きな数や小さな数をミスなく扱う。
* **関数電卓（EL-509T）のENG表示機能**　をマスターし、接頭語（k, m, μなど）への読み替えに慣れる。
* 数値を直接打ち込むのではなく、**「変数」**　に代入して公式（$V = I \times R$ など）をコードで表現する。

---

## 1. なぜ「3桁区切り」なのか？

エンジニアの世界では、$0.000047$ のような「0」が並ぶ数字は使いません。数え間違い（ヒューマンエラー）を防ぐため、指数を「3の倍数」で揃えるのが鉄則です。

**例**
> 123000 　→ 　$123 \times 10^3$   
> 0.0005 　→ 　$0.5 \times 10^-3$

Python では、`E`　または 小文字の `e` を使って表記します。

> 123000 　→ 　`123e3`  (`1.23e5`)  
> 0.0005 　→ 　`5e-3`  (`5.0e-4`)

以下に工学でよく使われる接頭語を紹介します。

| 接頭語 (Prefix)  | 記号 | 指数 (10^n) | Python表記 | 数値の例             |
| :-------------- | :-: | :--------- | :--------- | :------------------ |
| テラ (Tera)     |  T  | $10^{12}$      | 1e12       | 1,000,000,000,000   |
| ギガ (Giga)     |  G  | $10^9$       | 1e9        | 1,000,000,000       |
| メガ (Mega)     |  M  | $10^6$       | 1e6        | 1,000,000           |
| キロ (kilo)     |  k  | $10^3$       | 1e3        | 1,000               |
| (基本単位)       |  -  | $10^0$       | 1e0        | 1                   |
| ミリ (milli)     |  m  | $10^-3$      | 1e-3       | 0.001               |
| マイクロ (micro) |  u  | $10^-6$      | 1e-6       | 0.000001            |
| ナノ (nano)     |  n  | $10^-9$      | 1e-9       | 0.000000001         |
| ピコ (pico)     |  p  | $10^-12$     | 1e-12      | 0.000000000001      |

> **💡 工業のルール：** > 指数は必ず「3の倍数」を使います。  
> × 不適切： $1.2 \times 10^4$ （`1.2e4`）  
> ○ 適切　： $12 \times 10^3$ （`12e3` = 12k）  
> これにより、計算結果を見た瞬間に「12キロ」と判断できます。

## 2. 関数電卓（EL-509T）の活用

計算結果が `0.00047` と出たとき、パッと「何マイクロか」答えるために、EL-509Tの以下の機能を使いましょう。

* 　**[ALPHA] → [1]** ($10^n$)： 指数を3桁ずつ増やして表示（k → M → G...）
* 　**[ALPHA] → [2]** ($10^{n-1}$)： 指数を3桁ずつ減らして表示（m → μ → n...）

**【実習】**  
電卓で `0.00047` [=] と入力した後、[ALPHA][2] を押してみましょう。表示が `470. x 10^-6` （470μ）に変わることを確認してください。

## 3. 変数（Variable）を使って「公式」を書く

Python でプログラムを書くときは、数値を直接計算せず、一度「変数（名前の付いた箱）」にしまっておくと読みやすくなります。
これを 「**代入**」といいます。

### 良い例（公式 $V = I \times R$ がそのまま見える）

```python
# 1. 物理量を変数に入れる（指数表記を活用）
current = 4.5e-3      # 電流 I = 4.5 mA
resistor = 2.2e3      # 抵抗 R = 2.2 kΩ

# 2. 公式をそのまま書く
voltage = current * resistor

# 3. 結果を表示する
print(voltage)
```
---

## 💡 入力時の注意点

1. 大文字・小文字の区別
   - 単位の m（ミリ）と M（メガ）は別物です。
   - Pythonでは 1e3 も 1E3 も同じですが、小文字 e が一般的です。
   
    - Python で間違いやすい書き方

      | 誤った書き方 | 誤った点| 正しい表記 |
      | :---------: | :----- | :-------- |
      | `e4`     |  前の部分 (仮数部) が 1 でも省略できない  | `1e3`| 
      | `1.23e0.3` | 後ろの部分 (指数部) は整数のみ | 表記不可|
      | `5.0e(-3)` | 後ろの部分にカッコは不要 | `5.0e-3` |
      | `2.3*e3` | 前の部分と後ろの部分の間の `*` は不要| `2.3e3`|
      |`5.4**3` | `**` はべき乗の計算式で、指数の定数ではない | 誤り |

2. 「0」の数え間違いを防ぐ
   - 0.000001 と書くより 1e-6 と書くほうがミスを防げます。

3. 関数電卓 (EL-509T) の活用
   - 計算後 [ALPHA] > 1 または [ALPHA] > 2 ボタンを押すと、この3桁区切りの表示に切り替わります (エンジニア表示)。
