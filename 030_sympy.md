## 1. 専門家の道具箱「ライブラリ」を使おう

第2回までの Python は、いわば「普通の電卓」でした。複雑な因数分解や方程式を解くには、**SymPy（シムパイ）** という「数学専門のライブラリ」を呼び出す必要があります。

* **ライブラリとは？** : 特定の目的（数学、グラフ、AIなど）のために作られた、便利なプログラムの詰め合わせです。
* **使い方** : プログラムの先頭で `import` 命令を使い、道具箱を開きます。

```python
import sympy as sp  # sympyを「sp」という短い名前で呼ぶことにする
```

## 2. 文字式（Symbol）の準備

数学の公式のように、文字を文字のまま扱うためには、まず「これは数学用の文字（シンボル）ですよ」と教える必要があります。

```python
# 「x」という文字を定義する
x = sp.symbols('x')

# 文字式のまま計算し、きれいに表示（display）する
expr = (x + 1) * (x - 1)
display(expr)
```

> **💡 display() の魔法**
> `print()` ではなく `display()` を使うと、Google Colab 上で教科書のようなきれいな数式が表示されます。

## 3. 数式の整理（展開・因数分解）

教科書の第1章に出てくる「式の展開」や「因数分解」は、Python に任せれば一瞬で答え合わせができます。

### 3.1 展開（expand）
$(x + 2)^3$ のような面倒な計算をバラバラにします。

```python
expr = (x + 2)**3
sp.expand(expr)  # 結果： x**3 + 6*x**2 + 12*x + 8
```

### 3.2 因数分解（factor）

バラバラの式をカッコ同士の掛け算の形にまとめます。

```python
expr = x**2 + 5*x + 6
sp.factor(expr)  # 結果： (x + 2)*(x + 3)
```

---

## 4. 方程式を解く（solve）

ここがエンジニアとして最も強力な武器になります。

### ⚠️ 【重要】「＝」の使い分けに注意！

ここが一番間違いやすいポイントです。

1.  **`=` (代入)** : 変数に値を覚えさせるための記号。
2.  **`sp.Eq(左辺, 右辺)` (等式)** : 数学的な「イコール (等しい)」の関係を表す記号。

**例： $x^2 - 4 = 0$ を解く**

```python
# 1. 式を立てる： Eq(左辺, 右辺)
eq = sp.Eq(x**2 - 4, 0)

# 2. 解く： solve(式)
answer = sp.solve(eq)
print(answer)  # 結果： [-2, 2]
```

---

## 5. 【応用】公式の変形（等式の変形）

エンジニアは、状況に合わせて公式の形を書き換える必要があります。
SymPy はこれも得意です。

**問題：オームの法則 $V = I \times R$ を「$I$ について解く（$I = \dots$ の形にする）」**

```python
# 複数の文字を定義
v, i, r = sp.symbols('v i r')

# 公式を定義： V = I * R
ohm_siki = sp.Eq(v, i * r)

# i について解く
i_siki = sp.solve(ohm_siki, i)
display(i_siki) # 結果： [v/r]
```

---
## 6. 値の計算（subs）

解いた式に、具体的な数値を当てはめて数値を計算します。

```python
# 複数の文字を定義
v, i, r = sp.symbols('v i r')

# 公式を定義： V = I * R
ohm_siki = sp.Eq(v, i * r)

# i について解く
i_siki = sp.solve(ohm_siki, i)
display(i_siki) # 結果： [v/r]

v = ohm_siki.subs(i, 100e-3) # 100 mA
display(v)

v = ohm_siki.subs(r, 5e3) # 5 KΩ
display(v)

v = ohm_siki.subs( {i: 100e-3, r: 50e3} ) # 電流と抵抗を一度に当てはめる
display(v)
```

## 7. 注意点：SymPy でよくやるミス
| 誤った書き方 | 理由 | 正しい表記 |
| :--- | :--- | :--- |
| `x + 1 = 0` | `=` は代入なので方程式にならない | `sp.Eq(x + 1, 0)` |
| `2x` | 文字と数字の間の `*` を省略できない | `2 * x` |
| `x^2` | `^` は数学用ではない | `x ** 2` |
| `solve(x+1)` | 右辺を省略すると自動的に `= 0` とみなされる | `sp.Eq(x+1, 0)` と書くのが安全 |
