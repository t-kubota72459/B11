## 1. 言葉の意味を整理しよう

数学の用語は難しく聞こえますが、エンジニアの作業に例えるとシンプルです。

### 展開（expand）とは：ユニットの分解
因数分解された式や数を元に戻す（掛け算を計算する）ことを **展開する** と言います。
* **イメージ**：組み立てられた製品をバラバラに分解して、部品（項）を一つずつ並べる作業です。
* **いつ使う？**：式全体の中で、どの部品が一番大きな影響を与えているか分析するときに使います。

### 因数分解（factor）とは：部品をユニットにまとめる

数や式を「因数（掛け算のパーツ）」に分解することです。
たとえば $6 = 2 \times 3$ ですし、$100 = 2^2 \times 5^2$ と分解できます。このとき $2$ や $3, 5$ を **因数** と呼びます。
* **イメージ**：バラバラの部品を「( )」という箱に入れて、スッキリと整理整頓する作業です。
* **いつ使う？**：式が **$0$ になる条件（システムの停止や均衡点）** を一瞬で見抜きたいときに使います。

## 2. SymPy で公式を確認する

「和と差の積の公式」 $x^2 - y^2 = (x + y)(x - y)$ を使って、Python での操作を確認しましょう。
式には 2 つの記号 ($x, y$) があるので、Symbol を 2 つ同時に作ります。

```python
import sympy as sp
x, y = sp.symbols('x y')   # x, y を同時に定義

# 1. 因数分解してみる
siki = x**2 - y**2
bunkai = sp.factor(siki)      # siki を因数分解せよ
print("因数分解（整理整頓）の結果：")
display(bunkai)

# 2. 展開して元に戻してみる
tenkai = sp.expand(bunkai)    # bunkai を展開せよ
print("展開（バラバラに分解）の結果：")
display(tenkai)
```

## 3. なぜ因数分解が「急所」を見つけることになるのか？

エンジニアにとって、式が $0$ になる瞬間を知ることは非常に重要です。

**例**：暴走するモーターを止めるには？ $N$ が $N = I^2 - 7I +10$ という式で表されるとき
* **因数分解後**： $N = (I - 2)(x - 5)$ となります。

これを見れば、**「$I=2$ または $I=5$ になったとき、装置の出力が $0$ になる」** ということが、計算しなくても一目でわかりますね。これが因数分解の最大のメリットです。

```python
import sympy as sp

N, I = sp.symbols('N I')            # N と I を使う
shiki = sp.Eq(N, I**2 -7*I + 10)    # 式を作って
ans = sp.factor(shiki)              # factor... 因数分解
display( ans )                      # 掛け算になってる
```
