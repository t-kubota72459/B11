## 【課題1】品質管理とド・モルガンの法則（第5回の復習）

工場で生産された5つの部品（`Part_A` 〜 `Part_E`）の検査を行います。
出荷できる「合格品」の条件は、**「寸法が規格内である」 かつ 「表面に傷がない」** ことです。
ド・モルガンの法則（第2法則）を使って、出荷してはいけない **「不良品」** を洗い出すプログラムを完成させてください。

* **ド・モルガンの法則（第2法則）**： $\overline{A \cap B} = \overline{A} \cup \overline{B}$
 
  * 左辺：「寸法OK かつ 傷なし」**ではない**（合格の否定）
  * 右辺：「寸法NG」**または**「傷あり」

以下のコードの `___` の部分を正しい Python の記号（`&`, `|`, `-`）に書き換えて、左右の計算結果が一致することを確認してください。

```python
# 全部品のリスト（全体集合 U）
U = {'Part_A', 'Part_B', 'Part_C', 'Part_D', 'Part_E'}

# 検査をクリアした部品のリスト
sunpou_ok = {'Part_A', 'Part_B', 'Part_D'}    # 寸法が規格内の部品 (集合A)
kizu_nashi = {'Part_A', 'Part_C', 'Part_D'}   # 表面に傷がない部品 (集合B)

# 1. ド・モルガンの法則の「左辺」で不良品を計算する
# 合格品（寸法OK "かつ" 傷なし）を求め、全体 U から引く
hidari = U ___ (sunpou_ok ___ kizu_nashi)
print("左辺の不良品リスト:", hidari)

# 2. ド・モルガンの法則の「右辺」で不良品を計算する
# 「寸法NG（Uからsunpou_okを引く）」 "または" 「傷あり（Uからkizu_nashiを引く）」
migi = (U ___ sunpou_ok) ___ (U ___ kizu_nashi)
print("右辺の不良品リスト:", migi)

# 3. 法則が成り立っているか確認
print("法則は成り立つか？:", hidari == migi)
```

## 【課題2】NC工作機械の異常検知（第6回の復習）

最新のNC工作機械は、複数のセンサーの状態を「1つの2進数（8ビット）」にまとめて送信してきます。
各桁（ビット）の「1」は、以下の状態を表しています。

* **右から1桁目**： 電源ON
* **右から3桁目**： モーター回転中
* **右から5桁目**： 安全扉（カバー）が開いている ⚠️危険
* **右から8桁目**： 非常停止ボタンが押されている ⚠️危険

※マスク（フィルター）を作るときは、調べたい桁だけを「1」にし、**使っていない（調べない）桁はすべて「0」** としてください。

現在、機械から送られてきた状態信号は **`0b10010101`** です。
ビットマスク（AND演算）を使って、**「安全扉が開いているか」** と **「非常停止が押されているか」** を抽出するプログラムを完成させてください。

```python
# 現在の機械の状態信号
joutai = 0b10010101

# --- 調査1：安全扉は開いているか？ ---
# 右から5桁目だけを「1」にし、他を「0」にしたマスクを作る
tobira_mask = 0b00010000

# AND演算(&)で安全扉の桁だけを抽出する
tobira_check = joutai & tobira_mask
print("安全扉の確認（0b10000なら開いている）:", bin(tobira_check))


# --- 調査2：非常停止ボタンは押されているか？ ---
# 右から8桁目だけを「1」にし、他を「0」にしたマスクを作ってください
hijou_mask = 0b________  # ← ここを正しい2進数に書き換える

# joutai と hijou_mask を AND演算(&) して抽出してください
hijou_check = ______ & ______  # ← ここに正しい変数名と記号を入れる

print("非常停止の確認（0b10000000なら押されている）:", bin(hijou_check))
```
